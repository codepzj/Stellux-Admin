# 网站设置管理

## 功能概述

网站设置管理功能允许管理员自定义网站的主页和About页面的内容，无需修改代码即可更新网站展示的信息。

## 主要功能

### 1. 设置管理

- **创建设置**: 为主页或About页面创建新的设置
- **编辑设置**: 修改现有设置的内容
- **删除设置**: 删除不需要的设置

### 2. 主页设置

支持配置以下内容：

- 基本信息：标题、描述、姓名、头像
- 个人简介：个人介绍、名言
- 社交链接：GitHub、博客地址
- 位置信息
- 技术栈：展示的技术栈标签
- 开源项目：展示的开源项目列表
- 文章展示：是否显示最近文章及数量

### 3. About页设置

支持配置以下内容：

- 技能：按分类组织的技能列表
- 时间线：个人成长历程的时间线
- 兴趣爱好：兴趣标签
- 当前专注：正在专注的事项

## 使用说明

### 1. 访问管理页面

在管理后台的侧边栏中找到"网站设置"菜单项。

### 2. 创建设置

1. 点击"新建"按钮
2. 选择设置类型（主页/About页面）
3. 填写相应的配置信息
4. 保存设置

### 3. 编辑设置

1. 在设置列表中点击"编辑"按钮
2. 修改设置内容
3. 保存更改

### 4. 删除设置

- 点击"删除"按钮确认删除
- 删除后无法恢复，请谨慎操作

## API接口

### 管理接口

- `POST /admin-api/config/create` - 创建设置
- `PUT /admin-api/config/update` - 更新设置
- `DELETE /admin-api/config/:id` - 删除设置
- `GET /admin-api/page/list` - 获取配置列表
- `GET /admin-api/page/:id` - 获取配置详情

### 前端接口

- `GET /page/config/:type` - 获取激活的页面配置

## 注意事项

1. **数据验证**: 所有必填字段都会进行验证，请确保填写完整
2. **实时生效**: 设置更改后，前端网站会实时显示最新内容
3. **备份重要**: 删除设置前请确认不再需要，删除后无法恢复

## 技术实现

- **后端**: Go + Gin + MongoDB
- **前端**: Vue 3 + TypeScript + Ant Design Vue
- **数据存储**: MongoDB集合 `config`
- **缓存**: Next.js ISR缓存机制
